#Table Tennis America (TTA)
An iOS app built with SwiftUI and designed in Figma to help players engage with local table tennis tournaments.

Table Tennis America (TTA) gives players an easy way to browse schedules, register for matches, and track rankings—all in one place. 

### Tournament Schedules
- View upcoming matches and event times  
- Browse full tournament structures  
- Quickly find divisions or events of interest

### Match Registration
- Register for matches directly through the app  
- No paperwork or confusing forms  
- Instant confirmation for registered events

### Player Rankings & Stats
- Track personal performance and progress  
- View match history and results  
- Follow national and club-level standings

### Detailed Tournament Information
- Access rules, formats, and event descriptions  
- See player lists and division details  
- Stay updated with real-time in

## Tech Stack

- **SwiftUI** — Declarative UI framework  
- **Figma** — UI/UX design  
- **Xcode** — Development + testing
